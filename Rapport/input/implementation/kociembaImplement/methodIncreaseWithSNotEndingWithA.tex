\section{Method Increase With S Not Ending With A}

The method "Increase With S Not Ending With A" is the method that increments a move sequence. When the method is called it takes two parameters, a move sequence of the type MoveButtons and an integer. As the method initializes it creates two variables of the type integer, \textit{length} and \textit{i}. The variable \textit{length} is set equal to the length of the move sequence, so every time the length of the move sequence is needed the variable \textit{length} will be used. The variable \textit{i} is used to keep track of where in the move sequence a move is changed.
